# fix-modal-select

A Codex skill for fixing dropdown and select components that escape modal boundaries.

## What it captures
- Shared wrapper pattern for Reka or shadcn style select components
- `portal` opt-in and opt-out behavior
- Dialog-layer verification checklist

## Install
1. Clone or download this repository.
2. Copy the repository folder into `~/.codex/skills/fix-modal-select`.
3. Start a new Codex session.
